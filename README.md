# Automating Enclosure and Robot Arm Generation in FreeCAD
Welcome to my FreeCAD project.    
This repository contains the workbench that automates the generation of:
- Custom electronics enclosures
- Raspberry Pi Zero 2 and Raspberry Pi 4B enclosures
- HDMI, USB-C, USB-B cutouts
- Robot arm base and links

The scripts were written using either the Part workbench alone employing constructive solid geometry (CSG) modelling or a combination of the Sketcher and Part Design workbench (parametric modelling).
I will confess, the best ideas tend to arrive at the final stages of the project. I'm hoping to build on my work and introduce a script that can automatically detect the edges of a PCB imported from KiCAD and generate the enclosure according to the shapes it detects along the edges. I'm also looking forward to learning more about robotics so that I can improve the robotics features and make them flexible and relevant to more types of users.
